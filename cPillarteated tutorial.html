COMO CRIAR O CPILLARTEATED

No arquivo cPillarteated.html tem um código de Python, copie e cole em um compilador Python para ter o cPillarteated

COMO ABRIR UM ARQUIVO

Na linha arquivo = open('cp.cpa') altere o 'cp.cpa' para o nome e a extensao do seu arquivo

Os dois arquivos (O codificado e o cPillarteated) tem que estar na mesma pasta

COMO USAR O CPILLARTEATED

Clique no 'Run' na parte de cima e salve o cPillarteated para rodar

Não compartilhe links maliciosos nem virus usando o cPillarteated

Não use ele para piratear coisas, isso é errado!

O cPillarteated foi feito para compartilhar arquivos de texto anonimamente

Por mais que 'Pillarteated' pareça 'Pirated', em inglês 'Pirateado', não use o cPillarteated para crimes!

Os formatos dos arquivos normalmente são coisas aleatórias e impossíves de usar com um abridor de arquivos comum
